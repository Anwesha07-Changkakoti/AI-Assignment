# AI Assignment

Overview

This repository contains implementations of three important Artificial Intelligence algorithms developed in Python.

The programs demonstrate:

1. Hidden Markov Model (HMM)
2. Alpha-Beta Pruning
3. Tic-Tac-Toe using Minimax Algorithm

 1. Hidden Markov Model (HMM)
 Description

A Hidden Markov Model is a statistical model used to represent systems where the actual states are hidden but observations are visible.

Features

* User-defined observation sequence
* Hidden state prediction
* Demonstrates probabilistic reasoning

Applications

* Speech Recognition
* Natural Language Processing
* Bioinformatics
* Activity Recognition

2. Alpha-Beta Pruning
Description

Alpha-Beta Pruning is an optimization technique for the Minimax algorithm used in game playing.
 Features

* User-defined tree depth
* User-defined leaf node values
* Displays visited nodes
* Shows pruning of unnecessary branches
* Computes optimal game value

Applications

* Chess Engines
* Tic-Tac-Toe
* Checkers
* Strategic Decision Making

3. Tic-Tac-Toe using Minimax Algorithm

 Description

This project implements a Tic-Tac-Toe game where the computer uses the Minimax algorithm to determine the best possible move.

Features

* Human vs AI gameplay
* Intelligent move selection
* Optimal decision making
* Interactive console interface

Applications

* Game AI
* Decision Making Systems
* Adversarial Search Problems


Technologies Used

* Python 3
* Object-Oriented Programming Concepts
* Artificial Intelligence Algorithms
